Width y Height

Estas propiedades sirven para dar el ancho (Width) y el alto (Height) de los elementos que queramos aplicar. No en todos los casos se pueden usar ya que estos solo sirven para los elementos de bloque y no para los elementos lineales ya que el tama単o no determina su contenido. Veremos unos ejemplos sencillos para poder ver como funcionan.

Al elemento de bloque le vamos a dar una propiedad de width de 250px al poner el cursor sobre el elemento de bloque veremos cual es el espacio que representa dichos pixeles.

Elemento de bloque
Width 250px

En el siguiente ejemplo le aplicamos una propiedad de Height 250px.

Elemento de bloque
Height 250px

Como pudimos darnos cuenta los elementos de bloque al dar los atributos cambiamos unicamente el alto y el ancho. Si queremos dar otras propiedades de tama単o debemos usar Padding o Margin que veremos en otro segmento.

A los elementos de linea como los enlaces (links) las propiedades de Width y Height no funcionan ya que no tienen medidas y el tama単o lo determina su contenido por ejemplo:

padding
El fondo color naranja va hasta donde estan las letras

Entre mas largo sea el elemento de linea el fondo queda condicionado a su contenido

padding, descripcion y sus ejemplos
El fondo color azul claro abarca todo el contenido

Si queremos dar un tama単o al contenedor de los elementos de linea podemos usar la propiedad de Padding ya que este lo que hace es agrandar la caja donde esta el link, para saber mas de esta propiedad podemos ir al enlase y ver como funciona.
